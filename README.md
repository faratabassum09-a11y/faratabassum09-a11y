Full-stack LMS built with Node.js, Express.js, MongoDB following MVC architecture. Features role-based authentication (Admin/Student) via Passport.js, automated test scheduling with node-cron, Excel bulk upload via SheetJS, and PDF report generation via PDFKit. Deployed on Render with MongoDB Atlas.

`Node.js` `Express.js` `MongoDB` `Passport.js` `MVC` `Cron Jobs` `PDFKit` `SheetJS`

`Node.js` `Express.js` `MongoDB` `Cloudinary` `Bootstrap`
