`React` `Redux Toolkit` `Node.js` `Express.js` `MongoDB` `Socket.IO` `Google Gemini API` `Passport.js` `Cloudinary` `Vercel` `Render`

### 🎓 [Flashcard LMS — Learning Management System](https://flashcard-lms.onrender.com)
Full-stack learning management system built with Node.js, Express.js, and MongoDB using strict MVC architecture, with role-based workflows for Admin and Student users via Passport.js session authentication. Features a cron-driven automated test engine (node-cron) that starts and closes timed assessments on schedule with question shuffling and keyword-based evaluation, removing the need for manual proctoring. Includes Excel-based bulk student onboarding (SheetJS, Multer) and automated PDF score report generation (PDFKit).

`Node.js` `Express.js` `MongoDB` `Passport.js` `MVC` `node-cron` `PDFKit` `SheetJS`

### 🏠 [LuxeNest — Property Rental Platform](https://luxenest-gwil.onrender.com)
Full-stack property rental marketplace built with Node.js, Express.js, and MongoDB, featuring listing management, a review system, and session-based, role-scoped authorization. Integrated Cloudinary for cloud-based image storage with relational-style Mongoose schemas across listings, users, and reviews. Delivered a responsive, mobile-first UI with Bootstrap.

`Node.js` `Express.js` `MongoDB` `Cloudinary` `Bootstrap`
